units

css unit __ css hossaning o'lchovini bildiradi

unitlar 2xil bo'ladi:
absolute mutloq o'lchov birliklari qanday ko'rsatilgan bo'lsa shunday ekranga chiqadi bularni ishlatish tavsiya etilmaydi (ishlatiladigan joyi bor) masalan "px sm mm in"


relativ nisbiy o'lchov birliklari boshqa uzunlik o'lchoviga nisbatan o'z uzunligin belgilaydi turli xil ekranlarga moslashish uchun masalan: "% em rem"


% __ o'lchovi elementning o'lchovlarini parent elementning o'lchovlariga moslashtiradi agar parentida o'lchov (font-size) bo'lmasa parentning parentiga murojaat qiladi agar parentning o'lchovi o'zgarsa childning ham o'lchovi parentga moslashgan tarzda o'zgaradi masalan:
parent{
width:400px; }
child{ width:80%; }


em__ o'lchov birligi font-sizega qarb hisoblanadi agar elementning font-size hossasi bo'masa u parentining font-size qiymatiga asoslanadi. agar unda ham bo'lmasa keyingi ajdodining font-size qiymatiga asoslanadi.


rem __ root elementning font-size qiymatiga qaraydi root __ yani eng katta ajdod (ildiz) html
